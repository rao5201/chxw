\# 茶海虾王 - 电商综合平台



这是“茶海虾王”电商网站的官方开源代码仓库。本项目旨在展示一个完整的电商网站前端架构。

\## 📦 核心功能



\*   \*\*商品展示\*\*：支持茶叶与海鲜的图文展示。

\*   \*\*云端连接\*\*：前端已集成 Supabase 数据库连接逻辑。

\*   \*\*响应式设计\*\*：适配手机、平板和电脑端浏览。

\*   \*\*后台管理\*\*：提供简易后台界面用于商品管理。



\## 🛠️ 技术栈



\*   \*\*前端框架\*\*: HTML5 + Tailwind CSS

\*   \*\*交互逻辑\*\*: Vanilla JavaScript

\*   \*\*数据库\*\*: Supabase (PostgreSQL)

\*   \*\*部署\*\*: GitHub Pages
